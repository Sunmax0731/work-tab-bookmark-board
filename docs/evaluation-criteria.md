# Evaluation Criteria

QCDS は Quality、Cost、Delivery、Satisfaction と定義する。

| Grade | 基準 |
| --- | --- |
| S+ | 自動テスト、代表シナリオ、metrics、baseline、docs ZIP、manual test docs、security/privacy、traceability がそろう |
| S- | 上記の大半がそろい、残作業が軽微 |
| A+ | 実装と主要 docs は十分で、手動確認の一部が残る |
| A- | 公開前判断は可能だが、磨き込み余地がある |
| B+以下 | 公開前の証跡または手順に不足がある |

A- 未満の場合は同じブランチで改善し、再評価する。
