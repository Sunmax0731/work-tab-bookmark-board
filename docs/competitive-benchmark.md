# Competitive Benchmark

## 比較対象

- Chrome tab groups
- OneTab
- Raindrop.io
- bookmarks manager

## 採用する評価基準

- 入力から検証、証跡、再実行までが1つの流れであること。
- 手動確認が必要な項目を warning として残せること。
- 公開前に docs、release checklist、security/privacy を確認できること。
- タブ、用途メモ、プロジェクト名を1カードで保存する。

## 差別化

この repo は Chrome拡張 Side Panel + storage API に閉じた最小実装だけでなく、QCDS、traceability、manual test、docs ZIP を含めた公開前の運用品質を製品の一部として扱う。
